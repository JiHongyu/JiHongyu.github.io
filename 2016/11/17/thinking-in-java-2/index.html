Thinking in Java (2). 面向对象

2016-11-17

前言

Java 入门第二弹，面向对象。这里介绍 Java 中 OOP 的关键点和难点。

正文

Java 中的包的概念和 C++ 中的名字空间的概念是类似的。Java 的包有层次概念，可以很好的与文件目录结构很好的协作起来，而 C++ 的名字空间是一个平面的概念。Java 在每个 .java 文件里面只能定义一个包，相当于该文件的保护伞。

一个 .java 文件只能定义一个 public class，每一个 class 都会生成对应一个 .class 文件。
