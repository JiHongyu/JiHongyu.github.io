Use Sublime 3

2016-12-04

前言

Sublime Text 3 文本处理神器。

快捷键篇

Ctrl+D 选词

Ctrl+L 选中整行

Ctrl+← 向左单位性地移动光标，快速移动光标。

Ctrl+→ 向右单位性地移动光标，快速移动光标。

Ctrl+/ 注释整行或选中内容

Ctrl+K

Ctrl+K+U 改为大写

Ctrl+K+L 改为小写
